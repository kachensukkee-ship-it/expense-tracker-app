# Project Blueprint

## Overview

This is a Flutter application for tracking expenses. It allows users to input transaction details (title, amount, type, and category) and send the data to a Google Sheet via a Google Apps Script.

## Features

*   **Expense Logging:** Users can log their income and expenses.
*   **Categorization:** Transactions can be categorized (e.g., Food, Travel, Salary).
*   **Google Sheets Integration:** Expense data is sent to a Google Sheet for easy tracking and analysis.
